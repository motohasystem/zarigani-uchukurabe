# ウチュークラベ

天体のサイズを比較できるWebアプリケーションです。

## 機能

- 地球を基準として、様々な天体のサイズを視覚的に比較できます
- 惑星、衛星、恒星など、14種類の天体を選択可能
- 天体は正確な直径の比率で表示されます
- 新しく追加した天体が自動的に画面中央に表示されます
- 美しい宇宙をテーマにしたデザイン

## 使い方

1. `index.html` をブラウザで開きます
2. 最初は地球が表示されています
3. ドロップダウンメニューから天体を選択して「追加」ボタンをクリック
4. 選択した天体が右側に表示され、自動的に中央に配置されます
5. 「リセット」ボタンで最初の状態（地球のみ）に戻ります

## 開発者向け情報

### 天体データの追加

`app.js` の `celestialBodies` オブジェクトに新しい天体を追加できます：

```javascript
newBody: {
    name: '天体名',
    diameter: 12345,  // km単位
    color: '#ffffff',  // 16進数カラーコード
    type: '天体の種類'
}
```

### スケーリング

アプリは自動的にスケールを調整し、最大の天体がキャンバスの高さの70%に収まるように表示します。
